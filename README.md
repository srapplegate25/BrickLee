# BrickLee

Landing page for BrickLee — real estate bookkeeping.

## Tech Stack

- Vite + React 18
- Tailwind CSS v3
- Lucide-React icons
- Formspree for contact form

## Deployment

This project is deployed via Vercel (auto-builds from GitHub on every push).

## Formspree Setup

The contact form posts to Formspree. To configure:

1. Get your form ID from https://formspree.io
2. Open `src/App.jsx`
3. Replace `YOUR_FORM_ID` in the `FORMSPREE_FORM_ID` constant near the top of the file

## Local Development (Optional)

If you ever want to run locally:

```
npm install
npm run dev
```
